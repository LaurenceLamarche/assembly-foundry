# Assembly

Custom AI agents built for businesses that refuse to settle for generic.

## What's Inside

- **Next.js 14** with static export for Vercel
- **Tailwind CSS** with custom brand colors
- **Framer Motion** for scroll animations
- **Fully responsive** design

## Development

```bash
npm install
npm run dev
```

## Deploy to Vercel

```bash
# Install Vercel CLI if needed
npm i -g vercel

# Login and deploy
vercel login
vercel --prod
```

Or connect your GitHub repo to Vercel for auto-deploys.
